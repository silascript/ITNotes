## <span id="rime_install">安装</span>

### Linux 下安装

Linux 下安装 Rime 有基于「IBus」框架和「Fcitx」框架两种。

基于 IBus 框架：

#### ArchLinux
```shell
pacman -S ibus-rime
```

#### Ubuntu
```shell
sudo apt-get install ibus-rime
```
