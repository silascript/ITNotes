# IT 书籍清单

---

> [!tip] 
> 
> 此清单依赖插件 [Dataview](../../NoteSoft/Obsidian/Obsidian_Plugins_Note.md#Dataview) 插件。

```dataview
Table without ID
	书名 ,
	原版书名,
	versions as 书籍版本,
	bookautor as 作者,
	translator as 译者,
	publication_date as 出版日期,
	cover as 封面
From ""
Where econtains(booksdb, "BooksList")
```
