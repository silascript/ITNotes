# CSS 布局笔记

---

## 流式布局

流式布局，也称为「百分比自适应布局」，它是一种等比例缩放的布局方式。

---

## FlexBox

FlexBox：弹性盒子模型，是一种**一维**的布局模型。

弹性盒子改进了块模型，既不使用浮动，也不会合并弹性盒容器与其内容之间的 [外边距](CSS_Note.md#margin)。

弹性盒子由容器、子元素和轴构成，并且默认情况下，子元素的布局方向与横轴的方向是一致。

## Grid

Grid 布局是**二维**布局。
