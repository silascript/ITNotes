## 示例

### 遍历

* 示例 1
```shell
# 遍历 data 目录下txt文件并备份，备份文件名称加上年月日为后缀
# 1.txt -> 1.txt_20230923
suffix=`date +%Y%m%d`
   
for f_temp in `find data/ -type f -name "*.txt"`
do
   echo "备份文件$f_temp"
 cp ${f_temp} ${f_temp}_${suffix}                                                                                 
done
```

* 示例 2
```shell
local exuid_arr=()

# 过滤掉空行及使用#注释的行
for line in $(cat $exlist_path | grep -v ^$ | grep -v ^\#); do
	# 把每行扩展的 uid 存储进数组中
	exuid_arr+=($line)
done

```

if [[ $# -eq 0 ]]; then
