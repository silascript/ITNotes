## 安装与配置

### SDK

要使用 Dart 语言就得装相关的 SDK，Dart 有两种：Dart SDK 和 [Flutter SDK](Flutter_Note.md)。Flutter SDK 是在 Dart SDK 基础构建的，除了 Dart SDK 外，还包含完整 UI 框架。
