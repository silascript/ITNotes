# IT 书籍清单

---

> [!tip] 
> 
> 此清单依赖插件 [Dataview](../../NoteSoft/Obsidian/Obsidian_Plugins_Note.md#Dataview) 插件。

```dataview
Table without ID
	书名 ,
	原版书名,
	版本,
	作者,
	译者,
	出版日期,
	ISBN,
	cover as 封面
From ""
Where econtains(booksdb, "BooksList")
```
