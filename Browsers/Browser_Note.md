## Vivaldi

### 自定义 UI 设置 

要对浏览器某些组件进行外观设置可以进行以下步骤操作：

1. 在地址栏输入：`vivaldi://experiments`，这是「实验性功能」设置
2. 勾选「Allow CSS modifictaions」
3. 在「设置」-「外观」中，「自定义 UI MOD」指定 CSS 目录

```css
/* 标签栏 */
.tab-position .tab {
  font-size: 18px
}
/* 地址栏 */
#app {
    font-size: 18px
}
/* 书签 */
.bookmark-bar {
    font-size: 18px
}

```
