## iFlow CLI

[iFlow CLI](https://cli.iflow.cn) 是一个基于终端的 AI 助手工具，可以当成 [Claude Code](#Claude%20Code) 的国内平替。

> [!tip] 
> 
> iFlow CLI 将于 **2026 年 4 月 17 日（北京时间）** 正式停止服务。转向 [Qoder](#Qoder) 平台。

## Qoder

[Qoder](https://qoder.com) 是阿里推出的一个面向真实软件的智能体编程平台

### Qoder CLI

> [!info] 
> 
> 这应该就是 [iFlow CLI](#iFlow%20CLI) 的后继者了！

安装：

* `curl -fsSL https://qoder.com/install | bash`
* `npm install -g @qoder-ai/qodercli`
