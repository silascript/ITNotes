## Functional Interface

 #函数接口

Functional Interface：函数接口，这是 [JDK8](Java_Note.md#JDK8) 提供新的接口。

「函数接口」是有且只有一个抽象方法的接口。

函数接口可以用 `@FunctionalInterface` 注解来标识函数接口。

但无论是否用 `@FunctionalInterface` 标识接口，只要此接口有且只有一个抽象方法，那此接口便是「函数接口」。
